### Controles

* ⬆️ **Flecha arriba:** Avanzar
* ⬇️ **Flecha abajo:** Retroceder
* ⬅️ **Flecha izquierda:** Mover a la izquierda
* ➡️ **Flecha derecha:** Mover a la derecha
* ⏯️ **Botón “Pausar/Reanudar”**: Detiene o reanuda el juego
* ▶️ **Botón “Iniciar Juego”**: Comienza la animación

### Elementos visuales

* **Carretera** con líneas animadas.
* **Nubes** en movimiento continuo.
* **Árboles** que se desplazan hacia abajo simulando el paso del entorno.
* **Sol** y cielo dinámico.

### Tecnologías usadas

* **HTML5** para la estructura.
* **CSS3** para el diseño y botones.
* **JavaScript + WebGL 2.0** para el renderizado gráfico y la lógica del juego.
